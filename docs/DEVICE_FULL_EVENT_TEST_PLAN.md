# 기기 전체 이벤트 테스트 계획 (구현 가능한 모든 상황)

기기에서 **구현된 모든 사용자 이벤트**를 빠짐없이 검증하기 위한 체크리스트입니다.  
모드 전환, 참고도/힌트 on·off, 홈 왕복, 대국 기력·클릭·버튼 등 **모든 이벤트**를 포함합니다.

---

## 1. 화면(목적지) 구분

| 코드 | 화면 | 진입 경로 |
|------|------|-----------|
| HOME | 홈 | 앱 최초 / 백버튼 또는 홈 버튼 |
| BOARD | 분석(바둑판) | 홈 → "분석" |
| PLAY | 대국 | 홈 → "대국" |
| OFFLINE | 오프라인 | 홈 → "오프라인" |
| MINIGAME | 미니게임 | 홈 → "미니게임" (현재 비활성) |
| SANDBOX | 샌드박스 | (홈에서 진입 가능 시) |
