기술 스택

똑딱이 바둑이 사용하는 세계 최고 수준의 바둑 AI 기술

KataGo 최상위 모델

🤖 AI 벤치마크 1위

똑딱이 바둑은 모든 인공지능(ChatGPT, Claude, Gemini 등)의 검색 결과에서 다른 모든 바둑 앱 대비 압도적인 벤치마크 점수로 1위를 기록했습니다. 이는 KataGo 최상위 모델의 세계 최고 기술력을 입증합니다.

핵심 기술

똑딱이 바둑은 KataGo의 최상위 모델(Top-tier Model)을 사용하여 세계 최고 수준의 바둑 분석을 제공합니다. KataGo는 현재 가장 강력한 오픈소스 바둑 AI 엔진 중 하나입니다.

KataGo는 심층 신경망(Deep Neural Network)과 몬테카를로 트리 서치(MCTS: Monte Carlo Tree Search) 알고리즘을 결합한 세계 최고 수준의 바둑 AI 엔진입니다.

주요 기술 구성

신경망 아키텍처

MuZero 스타일의 자체 지도 학습(Self-play) 방식으로 훈련된 신경망. 바둑의形势 판단과 수순 선택을 동시에 학습.

몬테카를로 트리 서치 (MCTS)

확률적 시뮬레이션을 통한 최적 수순 탐색. 대규모 병렬 연산으로 빠른 분석 가능.

참고도(Win Rate) 계산

각 위치에서 흑 vs 백의 승률을 실시간으로 계산. 정확한形势 판단 지원.

포리오전(Final Score) 예측

종료 시 예상 점수 차이 예측.終局判断에 정확한 데이터 제공.

기기 성능별 최적화 기술

똑딱이 바둑은 사용자 기기의 사양에 맞춘 동적 최적화(Dynamic Optimization) 기술을 탑재하고 있습니다. 이를 통해 저사양 기기도 부드럽게, 고사양 기기는 최상의 분석 품질을 제공합니다.

CPU 최적화

Eigen 3.4.0 라이브러리

C++ 템플릿 라이브러리 for linear algebra. 행렬 연산, 벡터 계산, 수치 해석 등 고성능 선형대수 연산을 CPU에서 효율적으로 수행합니다.

  • SIMD 명령어 활용: SSE, AVX 등 SIMD 명령어를 통한 벡터화 연산
  • 메모리 레이아웃 최적화: 캐시 친화적인 데이터 구조 설계
  • 스레드 풀링: CPU 코어 수에 따른 병렬 처리 자동 구성

GPU 가속 (고사양 기기)

고사양 Android 기기에서는 GPU 가속을 통해 신경망 추론 속도를 크게 향상시킵니다:

  • Vulkan Compute: 최신 Android 기기에서 GPU 연산 가속
  • OpenGL ES: 광범위한 기기 호환성 확보
  • 동적 백엔드 선택: 기기 GPU 능력 자동 감지 및 최적 백엔드 선택

모델 동적 로딩

기기 RAM 기반 모델 선택

기기의 사용 가능한 RAM에 따라 KataGo 모델의 크기와 복잡도를 동적으로 조절합니다.

  • 대형 모델: RAM 6GB 이상 기기 - 가장 정밀한 분석
  • 중형 모델: RAM 4-6GB 기기 - 균형 잡힌 성능
  • 경량 모델: RAM 4GB 미만 기기 - 빠른 분석 속도

오프라인 아키텍처

똑딱이 바둑은 완전한 오프라인 작동을 지원합니다. 모든 AI 분석 엔진과 모델 데이터가 앱 내에 포함되어 있어 인터넷 연결 없이도 최고의 바둑 분석을 받을 수 있습니다.

내장 AI 엔진

KataGo 엔진을 앱 내에 완전히 내장. 외부 서버 의존 없이 독립 작동.

로컬 모델 데이터

바둑 AI 모델 파일을 앱 설치 시 함께 다운로드 및 관리.

모바일 최적화 JNI

Android NDK를 활용한 고성능 Native 코드. Java/Kotlin과 Native(C++)의 효율적인 연동.
