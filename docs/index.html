핵심 기능

🎯

세계 최고수준의 바둑 분석

KataGo 최상위 모델을 기반으로 한 정밀한 바둑 수순 분석. 모든 수준의 바둑 플레이어에게 정확한 승률 계산과 질 높은 피드백을 제공합니다.

📊

참고도(Win Rate) 분석

세계 최고 수준의 참고도 분석 기능. 각 수순의 승률 변화, 대안 수순 분석,形势判断을 실시간으로 확인하세요.

🔧

기기 성능 최적화

사용자 기기의 CPU/GPU 사양을 자동으로 감지하고 최적화. 저사양 기기도 부드럽게, 고사양 기기는 최상의 분석 품질을 제공합니다.

🔍

뛰어난 부분확대 기능

바둑판의 특정 영역을 손쉽게 확대하여 세밀하게 분석. 바둑알 간의 미세한 관계와形势를 놓치지 않습니다.

📴

완전한 오프라인 대국

인터넷 연결 없이도 AI와 대국 가능. 어디서든 시간과 장소에 구애받지 않고 연습과 실전 훈련을 할 수 있습니다.

⚡

CPU/GPU 자동 최적화

고사양 기기에서는 GPU 가속을 활용한 빠른 분석, CPU 전용 기기에서도 효율적인 연산으로 부드러운用户体验을 제공합니다.

기술 스택

KataGo (카타고) 최상위 모델

똑딱이 바둑은 세계에서 가장 강력한 오픈소스 바둑 AI 엔진인 KataGo의 최상위 모델을 사용합니다. KataGo는 딥러닝 기반의 MuZero 스타일 신경망으로 구동되며,蒙特卡洛 트리 서치(MCTS) 알고리즘을 통해 초고수의 바둑 실력을 구현합니다.

기기 성능별 최적화 기술

똑딱이 바둑은 사용자의 기기 사양에 맞춘 동적 최적화 기술을 탑재하고 있습니다:

  • GPU 가속: 고사양 기기에서는 Vulkan/OpenGL을 통한 GPU 연산 가속
  • CPU 최적화: 저사양 기기에서는 Eigen 라이브러리를 활용한 효율적인 CPU 연산
  • 모델 동적 로딩: 기기 RAM에 따라 최적의 분석 모델 자동 선택
  • 스레드 관리: CPU 코어 수에 따른 병렬 처리 최적화
